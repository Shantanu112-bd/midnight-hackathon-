# FEATURES.md

Required features for the Midnight White Belt Bootcamp.

## Table Stakes (Milestone 1 & 2)

- **Local Development Environment**: Ability to run a Midnight node and indexer locally.
- **Wallet Funding**: Successful funding of the Lace Midnight wallet from a local/testnet faucet.
- **Smart Contract Deployment**: Deploying a `.compact` contract and generating the necessary managed artifacts.
- **Private State Interaction**: Performing transactions that update private state via ZK proofs.
- **Frontend Integration**: A functional UI (or CLI) that can initiate transactions and display contract state.

## Differentiators (Advanced)

- **ZK Proof Efficiency**: Optimizing witness logic for faster client-side proving.
- **Multi-user Interactions**: Contracts handling private data from multiple distinct users.
- **Complex Private Logic**: Implementing circuits beyond simple incrementing (e.g., loans, credit scores).
